# 如何贡献下一站个人视角

这里收保研、考研、出国、求职、间隔年、转向等发展路径的个人复盘。路径经验应优先写清过程和限制，而不是只写结果。

## 文件命名

```text
[year]-[name-or-alias]-[topic].md
```

匿名作者使用化名或主题名，不使用真实姓名。

## 写作边界

- 写清路径、年份、专业背景和关键约束。
- 不维护实时政策、院校名单或岗位信息，改为说明查证方式。
- 不提交未授权材料、私人联系方式、具体个人负面评价或未经证实传闻。
- 新增文章后更新本目录的 `index.md`。
