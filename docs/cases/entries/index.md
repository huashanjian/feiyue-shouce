# 案例库

这里收录已经整理入库的莞中人样本。当前阶段先搭好结构，真实案例会在完成授权和脱敏后逐步加入。

## 新增案例

新增案例时，优先创建文件夹页面：

```text
docs/cases/entries/[case-id]/index.md
```

案例图片和附件放在该案例自己的 `_assets/` 目录中。
