# 如何提交案例

案例是飞跃手册最核心的资产：它记录具体莞中人如何做选择、怎么走弯路、最后到了哪里。案例可以匿名，但必须真实、可脱敏、可公开。

## 这里收什么

- 志愿填报、专业选择、转专业、辅修等关键选择复盘。
- 保研、考研、出国、直博、间隔年等深造路径。
- 实习、求职、转行、入职适应等职业路径。
- 失败、调整、重建计划的经历。
- 对学校、专业、城市或路径的长期体验。

## 不收什么

- 未经授权的聊天记录、截图和个人隐私。
- 针对具体个人的负面评价。
- 未经证实的传闻、群内敏感内幕。
- 只适合 WPS 指北群内讨论的材料。

## 文件怎么放

优先使用文件夹页面：

```text
docs/cases/entries/[case-id]/index.md
docs/cases/entries/[case-id]/_assets/
```

`case-id` 使用英文小写连字符，例如 `2021-anonymous-sysu-eecs-recommendation`。

## 写作步骤

1. 从 [案例模板](templates/case-template.md) 或 [完整路径案例模板](templates/full-path-case.md) 复制结构。
2. 补齐 frontmatter，至少写明届别、大学、专业、路径、署名或匿名方式。
3. 写清背景、关键选择、时间线、过程、结果、复盘和不适合照搬的人。
4. 更新 [案例库](entries/index.md) 和相关索引页。
5. 提 PR，并确认公开转载和脱敏边界。

## 公开转载告知

合并到飞跃手册主站的公开内容，未来可能被莞言瓜语转载、摘编、排版或二次传播；维护者会尊重作者署名、匿名和脱敏约定。
