# 案例模板

案例模板用于把个人经历整理成可以公开阅读、可以被索引、也能长期维护的样本。

## 可用模板

- [通用案例模板](case-template.md)：适合大多数完整经历。
- [完整路径案例模板](full-path-case.md)：适合时间线清楚、跨度较长的样本。
- [访谈式案例模板](interview-style-case.md)：适合维护者根据访谈整理。

## 使用提醒

- 案例不是宣传稿，也不是匿名爆料箱。
- 写清背景和限制，比写一个漂亮结论更重要。
- 所有涉及个人、老师、同学、具体组织的内容都要先脱敏。
- 不适合公开的真实材料留在 WPS 指北，不进入仓库。
