# 完整路径案例模板

# 标题：从莞中到某条路径的完整复盘

> 一句话说明这条路径的起点、终点和最大变量。

## 基本信息

- 莞中届别：
- 大学入学年份：
- 本科院校与专业：
- 主要路径：
- 当前去向：
- 署名方式：

## 路径总览

用几句话概括这条路径的关键阶段，不需要写成励志故事。

## 高中到入学

记录志愿、专业、城市、家庭沟通等关键选择。

## 大学阶段

记录课程、绩点、科研、竞赛、实习、社团、失败和调整。只写与路径判断有关的内容。

## 申请或求职阶段

记录准备节奏、材料、面试、选择和结果。

## 复盘

写清楚哪些经验值得复用，哪些只是当时环境下的偶然结果。

## 不适合照搬的人

说明样本边界。
