# 如何贡献院校个人视角

这里收院校选择和就读体验的个人视角。学校体验容易受学院、校区、年份和个人路径影响，写作时请把背景说清楚。

## 文件命名

```text
[year]-[name-or-alias]-[topic].md
```

匿名作者使用化名或主题名，不使用真实姓名。

## 写作边界

- 写清大学、学院或专业、入学年份、校区和路径背景。
- 可以写优缺点，但不要写具体个人负面评价。
- 不把单个学院或宿舍体验写成整所大学的定论。
- 新增文章后更新本目录的 `index.md`。
