# 如何贡献中山大学个人视角

这里收中山大学的就读体验、专业选择、校园生活和路径复盘。个人视角应写清背景、限制和复盘，不把个体经历包装成普遍结论。

## 文件命名

```text
[year]-[name-or-alias]-[topic].md
```

匿名作者使用化名或主题名，不使用真实姓名。

## 写作边界

- 写清年份、专业、学校、路径或阶段背景。
- 不提交未授权截图、私人联系方式、具体个人负面评价或未经证实传闻。
- 不适合公开的真实材料留在 WPS 指北，不进入仓库。
- 新增文章后更新本目录的 `index.md`。
