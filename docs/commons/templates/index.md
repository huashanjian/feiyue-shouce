# 投稿模板

这里存放可以复制使用的公开内容模板。模板不是强制格式，而是帮助贡献者把背景、边界、署名和脱敏信息说清楚。

## 可用模板

- [通用页面模板](page-template.md)：适合方法论、资源页、板块说明。
- [个人视角模板](perspective-template.md)：适合校友经验、判断框架、局部复盘。
- [完整路径案例模板](../../cases/templates/full-path-case.md)：适合从高中到大学、升学、就业等完整路径样本。
- [访谈式案例模板](../../cases/templates/interview-style-case.md)：适合维护者根据访谈整理成文。

## 使用原则

- 模板里的字段可以按实际情况删减，但不要删掉署名、匿名、脱敏和公开转载边界。
- 不确定是否适合公开的材料，先不要放进仓库。
- 个人经历优先放进 `cases/entries/` 或对应 `perspectives/`，再由主干页面引用。
