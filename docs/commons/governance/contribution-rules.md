# 贡献规则

本页定义飞跃手册第一版的贡献边界。它服务两个目标：让校友愿意写一点，也让公开站点长期可靠。

## 内容边界

- 公开、可传播、已脱敏的内容进入飞跃手册主站。
- 真实但敏感、只适合非公开讨论的内容不进入公开仓库。
- 未确认、未授权、无法脱敏的材料不进入仓库。

## 提交要求

- 新页面使用英文小写连字符路径。
- 个人经验优先放入 `cases/entries/` 或对应 `perspectives/`。
- 新增页面后更新所在目录的 `index.md`。
- 不确定是否应进入主导航时，先只更新局部索引。
- 涉及图片和附件时，使用页面所在目录下的 `_assets/`。

## 脱敏要求

不要提交私人联系方式、未授权聊天记录、具体个人负面评价、未经证实传闻或群内敏感内幕。必要背景可以保留，但要改写到不指向具体个人。

## 公开转载告知

合并到飞跃手册主站的公开内容，未来可能被莞言瓜语转载、摘编、排版或二次传播；维护者会尊重作者署名、匿名和脱敏约定。
