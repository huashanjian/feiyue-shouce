# 如何贡献共同体故事

这里收与莞中人共同体有关、适合公开传播的故事和项目记录。它不收群内敏感讨论，也不替代个人案例库。

## 文件命名

```text
[year]-[name-or-alias]-[topic].md
```

如果涉及多人，请先确认署名和公开边界。

## 写作边界

- 写清故事发生的时间、背景和参与者展示方式。
- 不提交未授权照片、聊天记录或私人联系方式。
- 不把群内敏感材料整理到公网。
- 新增文章后更新本目录的 `index.md`。

## 公开转载告知

合并到飞跃手册主站的公开内容，未来可能被莞言瓜语转载、摘编、排版或二次传播；维护者会尊重作者署名、匿名和脱敏约定。
