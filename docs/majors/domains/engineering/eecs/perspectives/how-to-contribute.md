# 如何贡献经验观点

这里收电子信息与计算机专业群相关的个人经验，例如专业选择、课程路线、科研、实习、方向切换、读研或就业复盘。

## 文件命名

```text
[year]-[name-or-alias]-[topic].md
```

匿名作者使用化名或主题名，不使用真实姓名。

## 写作边界

- 写清学校、专业、年级和方向背景。
- 区分“这个学校这样安排”和“这个专业通常如此”。
- 不提交课程群截图、私人联系方式、具体个人负面评价或未经证实传闻。
- 新增文章后更新本目录的 `index.md`。
