# 如何贡献方向地图

这里收 EECS 方向地图的结构补充，例如方向入口、学习顺序、常见误解、代表课程、项目资源和莞中人经验链接。

## 文件命名

方向页优先使用文件夹页面：

```text
docs/majors/domains/engineering/eecs/fields/[field-name]/index.md
```

个人经验不要直接写进方向主干页，优先放到对应 `perspectives/` 或案例库。

## 写作边界

- 不把某个学校的培养方案写成整个方向的标准。
- 资源列表要说明适合谁，不做堆链接。
- 不提交盗版资料、课程群内容或私人联系方式。
- 新增方向后更新本目录的 `index.md`。
