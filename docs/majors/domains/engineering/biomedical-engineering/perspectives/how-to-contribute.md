# 如何贡献生物医学工程经验观点

这里收生物医学工程相关的个人经验、选择复盘、学习路线、方向判断和去向观察。个人视角应写清背景、限制和复盘，不把个体经历包装成普遍结论。

## 文件命名

```text
[year]-[name-or-alias]-[topic].md
```

匿名作者使用化名或主题名，不使用真实姓名。

## 写作边界

- 写清年份、学校、专业、方向和阶段背景。
- 区分“这个学校这样安排”和“这个专业群通常如此”。
- 不提交未授权截图、私人联系方式、具体个人负面评价或未经证实传闻。
- 新增文章后更新本目录的 `index.md`。
