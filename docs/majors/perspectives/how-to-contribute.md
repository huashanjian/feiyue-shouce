# 如何贡献专业个人视角

这里收专业选择、专业体验、转专业、相邻方向比较等个人视角。它不替代专业主干页，而是保留具体人的背景和判断过程。

## 文件命名

```text
[year]-[name-or-alias]-[topic].md
```

匿名作者使用化名或主题名，不使用真实姓名。

## 写作边界

- 写清你的届别、大学阶段、专业或方向背景。
- 可以写不适合谁，但不要攻击具体个人、课程教师或同学。
- 不把单个学校的体验写成整个专业的定论。
- 新增文章后更新本目录的 `index.md`。
