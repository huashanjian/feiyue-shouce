# 如何贡献志愿经验观点

这里收志愿填报相关的个人复盘和局部经验。它可以是一个判断方法、一次家庭讨论、一个信息源辨别过程，也可以是一段后来发现不该照搬的选择。

## 文件命名

```text
[year]-[name-or-alias]-[topic].md
```

匿名作者使用化名或主题名，不使用真实姓名。

## 写作边界

- 写清届别、分科或地区背景，避免读者误用。
- 不维护每年会变化的具体录取政策，改为说明如何查证。
- 不提交未授权截图、隐私信息、具体个人负面评价或未经证实传闻。
- 新增文章后更新本目录的 `index.md`。
