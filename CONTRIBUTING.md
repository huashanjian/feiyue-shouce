# 贡献指南

飞跃手册第一版以 Markdown 为主，使用 YAML frontmatter 记录元数据。

## 投稿入口

- 完整经历：[案例库](docs/cases/entries/index.md)
- 案例格式：[案例模板](docs/cases/templates/case-template.md)
- 局部经验：[贡献方式](docs/start-here/contribute/index.md)
- 共用代码：[注册表](docs/commons/registries/index.md)
- 投稿模板：[投稿模板](docs/commons/templates/index.md)

## 语言和公式

- 正文：Markdown
- 元数据：YAML frontmatter
- 行内公式：`$a^2+b^2=c^2$`
- 块级公式：`$$E = mc^2$$`

第一版不把完整 `.tex` 文件作为内容源。
